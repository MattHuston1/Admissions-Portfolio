Describe a Process

Let's say there's a lamp that won't turn on in the living room. The first thing I would do is check to see if it's plugged in. If it isn't I would plug it in and try again. If it is I would test the lamp on a different outlet and also test a seperate electronic device on both outlets. If the lamp turns on with the other outlet then I know that the problem is the first outlet. My current house has GFCI outlets so I would test and reset that outlet first. If the lamp doesn't work on either outlet while the other electronic device does work then I know that problem is either the lamp or the bulb. Next step is to replace the bulb. If a new bulb doesn't work and we know the outlet is good then it must be the lamp. If I really liked this lamp I would open it up and check if any of the wires came loose. If so I would grab my soldering iron and solder it back and then try again. If this step fails I believe it is time to throw away the lamp.
